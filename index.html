Flight briefing — read this first

This page is ready to fly, except for one problem: its two photos. The archive only hands out the originals — museum-grade files far too heavy for a web page. Your mission has three phases:

  1. Fetch the cargo. Download both originals from the sources listed in README.md and save them into the images/ folder under the exact names below.
  2. Weigh it. With the page open, use the browser's DevTools Network tab. Reload and read two numbers at the bottom: how many megabytes transferred and the finish time. Write them down.
  3. Trim it. Use Squoosh to resize each photo to at most 1200 pixels wide and export it. Replace the file under the same name. Then weigh again.

This page expects exactly two photos:

  • images/mars-selfie.jpg — Curiosity's self-portrait
